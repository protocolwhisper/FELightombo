# Lightombo Developer Tools Frontend

This project is a Next.js frontend application designed to replicate the Geomi developer platform experience, adapted for the Lightombo ecosystem. It features a modern, high-contrast design using Lightombo's signature yellow and black color palette.

## Project Structure

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Language**: TypeScript

## Getting Started

To run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:3000 with your browser to see the result.

## Design Decisions

- **Color Palette**: rigorous adherence to Lightombo branding (Neon Yellow #FAFF00, Black #000000, White #FFFFFF).
- **Typography**: Clean, sans-serif fonts for readability and modern aesthetic.
- **Components**: Modular React components for scalability (Hero, FeatureCard, Navbar, Footer).
- **Responsive**: Fully responsive layout using Tailwind CSS.

## Features

- **Hero Section**: Impactful landing area with clear call-to-actions.
- **Product Grid**: Detailed feature breakdown of developer tools (API Access, Indexing, Gas Station, NFT Studio).
- **Informational Sections**: Highlights of Movement network benefits and Lightombo tools (Performance, Security, Fees).
- **Navigation**: Intuitive navbar and footer for easy site traversal.
